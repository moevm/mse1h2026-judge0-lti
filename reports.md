# mse1h2026-judge0-lti
## О проекте
### Постановка задачи

Для IDE https://github.com/judge0/ide сделать интерфейсы подключения LTI и логику задач.

### Требования

- автоматизация деплоя IDE
- модуль для взаимодействия с moodle по LTI
- модуль для задач 
  - task_id и параметры (язык программирования, время на попытку … )
  - классы для задач
- измерения по ресурсам - сколько нужно ресурсов на одного пользователя системы, на запуск одной программы
- задачи-примеры

Технологии - JS, LTI, Python, Docker.
